# investment-banker

This project demonstrates work done for a banking company that meets functional requirements and provided standards along with pseudocode that was created. It focuses on Object Oriented Programming using secure and efficient code. The banking company wanted displays and calculations of an Initial Investment Amount, Monthly Deposit, Annual Interest (Compounded), the Number of Years, and a way to see the data. After user input the user will see the calculations and the interest after the Number of Years. Code effectively validates user input. Console contains all required elements. Charts display correct balances and earned interest without additional monthly input. Code uses industry standard best practices such as inline comments and approriate naming conventions to enhance readability and maintainability.
